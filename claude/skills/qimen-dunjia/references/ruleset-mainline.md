# `mainline-cn-v1` 默認規則

這個 skill 的正式排盤只使用這一套默認規則，不在正式計算裡混用別的說法。

## 適用範圍

- 面向華語圈（臺灣 / 中國大陸 / 港澳）常見時家轉盤奇門使用習慣
- 默認時區：`Asia/Taipei`（同 UTC+8，與北京、香港時間數值一致）
- 海外用戶必須先補城市或時區

## 固定規則

### 1. 時間規則

- 用戶說“現在”，就按當前絕對時間起局。
- 沒給時區且語境是臺灣 / 中國大陸 / 港澳，默認 `Asia/Taipei`（UTC+8）。
- 海外用戶沒給時區時，先追問，不要直接排盤。

### 2. 節令規則

- 以節令切換陰遁 / 陽遁。
- 冬至到芒種：陽遁
- 夏至到大雪：陰遁
- 節氣邊界前後 24 小時內，輸出中要明確寫出“邊界提醒”。

### 3. 三元規則

- 按日干支在六十甲子序列中的位置分組。
- 每 5 日為一元。
- 順序固定循環為：上元 -> 中元 -> 下元。

這是當前版本採用的工程化固定規則，用來確保同一輸入得到同一結果。

### 4. 定局規則

- 陰遁 / 陽遁與三元確定後，從固定定局表直接取局數。
- 不在正式排盤時臨時切換拆補法或其他變體。

### 5. 中宮規則

- 中宮相關判斷一律寄坤處理。
- 如果旬首所遁之儀或時乾落在中宮，正式輸出裡要提示“本盤按寄坤處理”。

### 6. 展示規則

- 不展示完整推理鏈。
- 只展示關鍵計算結果和必要依據。
- 用戶主要看到的是：陰陽遁、局數、旬首、旬空、值符、值使、關鍵用神。

## 不支持的情況

以下情況不進入正式排盤：

- 用戶明確要求切換到別的規則，但不接受當前默認規則
- 缺少時間或地點信息
- 想讓模型跳過腳本直接手算
