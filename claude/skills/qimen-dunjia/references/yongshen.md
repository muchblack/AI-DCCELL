# 用神取用順序

正式解盤不要一上來堆很多用神。先抓主問題，再確定主用神，最後再加輔助參考。

## 總順序

1. 先確認“誰是主體”
2. 再確認“要判斷什麼結果”
3. 先定主用神
4. 再加 1 到 2 個輔助參考

## 基礎定位

### 求測者本人

- 優先看日干落宮

### 所問之事 / 對方 / 當前動作

- 優先看時乾落宮

## 常見事項

### 事業 / 工作 / 項目推進

- 主用神：開門
- 輔助：值符、日干

先看：

- 開門得不得力
- 日干和開門關係如何
- 值符是否給到主導權和支持

### 求財 / 交易 / 投資

- 主用神：生門
- 輔助：戊、日干

先看：

- 生門有沒有被克
- 戊的位置是否穩
- 日干能不能接住這筆財

### 感情

先問清楚求測者的性別和對象性別，再確定用神：

- 男問女（或問異性）：乙為主要參考
- 女問男（或問異性）：庚為主要參考
- 同性關係：優先取六合為主用神，輔助看日干與時乾的關係
- 輔助：六合、日干

如果用戶沒有明說對象性別，先補問，不要擅自假設。

### 考試 / 學習 / 證書

- 主用神：景門、天輔
- 輔助：丁

### 疾病 / 健康

- 主用神：天芮、死門
- 輔助：日干

輸出時必須加現實提醒：

- 奇門只能看趨勢，不代替就醫和檢查

### 出行 / 方位 / 見人

- 主用神：目標方位所在宮
- 輔助：開門、生門、值符

### 訴訟 / 合同 / 爭議

- 主用神：開門、驚門
- 輔助：天心、值符

### 尋人 / 尋物

- 主用神：時幹 + 類象
- 輔助：玄武、門的狀態

## 解讀方法

寫用神分析時按這個模板：

1. 這次主用神我取什麼
2. 我為什麼這樣取
3. 它現在落在哪一宮
4. 這宮裡的門、星、神是幫它還是壓它
5. 這對用戶最關心的問題意味著什麼
