# 示例輸出

本文件展示兩類場景：正式排盤（含真實腳本輸出→解讀映射）和理論學習模式。

---

## 示例 1：正式排盤（跳槽）

### 訪談確認信息

- 事項：跳槽/換工作
- 起局時間：2026-03-24 10:30（公曆）
- 地點：上海（China/Asia/Shanghai）
- 目標：能不能動，什麼時候動更穩
- 輸出偏好：直接結論

### 腳本關鍵輸出（摘要）

執行腳本後從輸出 JSON 提取以下關鍵字段：

```
ganzhi:
  year:  丙午
  month: 辛卯
  day:   丁酉
  time:  乙巳
  time_xun:     甲辰
  time_xunkong: 寅卯

chart:
  dun_type:          陽遁
  yuan:              上元
  ju_number:         1
  xunshou:           甲辰
  hidden_yi:         壬
  kongwang:          [寅, 卯]
  kongwang_palaces:  [3, 8]  ← 震宮（東）、艮宮（東北）旬空
  time_stem_visible: 乙      ← 用戶代表

  zhifu:  { star: 天芮, palace: 9 }  ← 離宮（南）
  zhishi: { door: 死門, palace: 9 }  ← 離宮（南）

palaces（部分）：
  宮4 巽宮/東南：地盤辛 / 天盤乙 / 天英 / 景門 / 九天  ← 乙（用戶）所在
  宮7 兌宮/西：  地盤丁 / 天盤癸 / 天心 / 開門 / 太陰  ← 主用神開門所在
  宮9 離宮/南：  地盤乙 / 天盤己 / 天芮 / 死門 / 值符  ← 值符值使所在

warnings:
  - 本規則集中宮相關判斷一律寄坤處理。
```

### 讀盤過程（內部，不展示給用戶）

1. **主用神**：事業/跳槽 → 取開門為主用神，輔助看值符和時幹（乙）。
2. **開門落宮**：7宮兌宮（西），配天心/太陰。天心主聰明謀略，太陰主隱秘暗動 → 開門得神配合，但屬暗動、低調路線，不適合大張旗鼓。
3. **用戶代表（時幹乙）**：落 4宮巽宮（東南），配景門/九天 → 用戶目前視野開闊、動機強，但景門偏中等，執行力不是最強。
4. **值符值使**：同落 9宮離宮（南），死門 + 天芮 → 死門不利主動推進，天芮主遲滯/消耗，當前大格局偏保守。
5. **旬空**：寅卯對應 3宮（東）和 8宮（東北）→ 這兩個方向力量虛，今日往東或東北向談事效果打折。
6. **核心矛盾**：開門本身條件不差，但值使死門壓著，時機略顯保守；事不急則可動，急則容易在細節上受卡。

### 正式輸出（面向用戶）

---

**已確認信息**

- 看什麼事：跳槽，判斷能不能動、什麼時候動更穩
- 起局時間：2026-03-24 10:30，上海（亞洲/上海時區）
- 輸出方式：直接結論

**使用規則與默認項**

- 規則集：`mainline-cn-v1`（時家轉盤奇門）
- 時區：`Asia/Shanghai`（默認）
- 提示：本盤中宮寄坤處理

**盤面摘要**

- 陽遁 1 局
- 旬首：甲辰，旬首奇儀：壬
- 旬空：寅、卯（東方和東北方力量偏虛）
- 值符：天芮 · 離宮（南）
- 值使：死門 · 離宮（南）

**用神與關鍵依據**

主看開門（代表事業推進），輔助看值符和時幹。

- 開門落兌宮（西），配天心、太陰——天心主謀略周全，太陰主低調隱秘推進，說明這件事有條件做，但更適合私下接洽、不宜大張旗鼓。
- 你本人（時幹乙）落巽宮（東南），配景門，動力足、看法多，但還在觀望階段。
- 值使是死門，和值符同在離宮——說明當前大氣場偏保守，正面硬衝能量偏弱。

**核心判斷**

- 能不能動：可以動，但不適合用急攻的方式推進。
- 什麼時候動更穩：比起立刻明牌，更適合先私下探底（薪資、團隊、職責邊界），把條件談清楚再正式推進。
- 旬空提示：今日往東或東北方向談合作效果偏虛，若有面試安排可留意對方公司方位。

**方位 / 時機 / 行動建議**

- 優先選西方或西南方向的機會先接觸。
- 先做信息核實，再推進 offer 流程。
- 如果 offer 已在手，等條件談穩後再決定，不用這兩天就拍板。

**風險提醒與免責聲明**

此類決定還要結合真實 offer 條件、現金流、試用期風險綜合判斷。

## 示例 2：理論學習模式

### 場景

用戶說：值符和值使是什麼意思？

### 處理方式

不進入正式排盤。先確認學習需求，再解釋：

---

**值符**（zhífú）：當局的"主導星神"，代表當前時間段最有權威、最有主導力的力量方向。它的落宮決定了整個盤裡"誰說了算"，是判斷主動權和支持來源的關鍵參考。

**值使**（zhíshǐ）：當局的"執行門"，代表當前最活躍的行動通道。它的門名（休/生/傷/杜/景/死/驚/開）直接影響當前適合做什麼事。

**兩者關係：**

- 值符和值使通常一起看——如果兩者落在同一宮，說明"主導力量"和"行動通道"高度集中，效率高但也意味著其他宮位相對弱。
- 如示例 1，值符（天芮）和值使（死門）同在離宮（南），死門主消耗收尾，說明當前時間段不適合激進開拓。
