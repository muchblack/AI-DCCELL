## 基礎概念

四化是紫微斗數中最重要的動態機制，代表星曜受到天干的影響而產生質變。

### 四化含義

| 四化 | 符號 | 五行 | 核心含義 | 類比 |
|------|------|------|---------|------|
| 化祿 | 祿 | 土 | 財祿、順利、增加、得到 | 收入增加、機會來臨 |
| 化權 | 權 | 木 | 權力、掌控、努力、競爭 | 升職加權、主動爭取 |
| 化科 | 科 | 水 | 名聲、貴人、科甲、平順 | 聲名遠播、逢凶化吉 |
| 化忌 | 忌 | 水 | 糾纏、執著、不順、損失 | 困擾、挫折、放不下 |

### 四化的層次
- **生年四化（本命四化）**：影響一生，為先天格局的一部分
- **大限四化**：影響十年，以大限宮幹飛四化
- **流年四化**：影響一年，以流年天干飛四化
- **流月/流日四化**：更細的推演，用法相同

### 重要原則
- 化祿、化權、化科為三吉化，化忌為兇化
- 化忌不一定全兇，要看忌入何宮、忌何星
- 同一顆星同時化祿又化忌（不同層次），代表該領域有得有失
- 雙化祿、雙化權會加強效果，雙化忌則問題加倍

| 宮位 | 含義 |

### 化忌的深層理解
化忌不全是兇——它代表"在意""執著""課題"：
- 化忌入夫妻：很在意感情，因為在意所以糾結
- 化忌入官祿：非常重視事業，因為重視所以壓力大
- 化忌可以理解為"這輩子的功課"，是需要去修煉和成長的領域

## 運限四化

### 大限四化
以大限所在宮位的天干飛四化，疊加在本命盤上分析。

**大限四化的讀法**：
1. 找到當前大限的宮位天干
2. 用該天干查四化表
3. 看四化分別飛入本命盤的哪些宮位
4. 與本命四化疊加分析

**疊加效應**：
| 組合 | 含義 |
|------|------|
| 本命化祿 + 大限化祿 | 雙祿臨門，該領域大發 |
| 本命化祿 + 大限化忌 | 祿忌交戰，先好後壞或有得有失 |
| 本命化忌 + 大限化忌 | 雙忌疊加，該領域壓力極大 |
| 本命化忌 + 大限化祿 | 忌處逢祿，困境中有轉機 |
| 本命化權 + 大限化權 | 雙權，該領域權力極大但也有爭鬥 |
| 本命化科 + 大限化科 | 雙科，名氣大增 |

### 流年四化
以流年天干飛四化，疊加在大限和本命盤上。

流年命宮的確定：流年地支所在的宮位即為流年命宮，其餘十一宮按逆時針排列。

### 三層四化綜合分析步驟
1. 先看本命四化定先天格局
2. 疊加大限四化看十年運勢
3. 再疊加流年四化看當年狀況
4. 重點關注三層四化是否集中在同一宮位——集中之處必是該年焦點

## 飛星進階

### 宮幹飛四化（飛星派核心技法）

飛星派的精髓在於：不僅看生年四化，還關注每個宮位的宮幹所飛出的四化落入哪些宮位，由此建立宮位之間的因果關係。

**基本操作**：
1. 取某宮的天干
2. 用該天干查四化表
3. 看四化飛入哪些宮位
4. 分析"源宮"與"落宮"的關係

**舉例**：
如果夫妻宮天干為甲，甲年四化為：廉貞化祿、破軍化權、武曲化科、太陽化忌。
找到這四顆星分別在盤中的位置，就知道夫妻宮的"能量"飛到了哪裡。

**常用飛星斷語**：
- A宮飛祿入B宮 → A的資源和善意給了B
- A宮飛忌入B宮 → A給B帶來困擾或在B方面有執念
- A宮飛忌衝B宮（忌入B的對宮）→ A對B有破壞力
- 自化祿（A宮飛祿回到A宮）→ A宮內部自行生財
- 自化忌（A宮飛忌回到A宮）→ A宮自我糾結消耗

### 化忌追蹤法
在一個盤中追蹤化忌的走向是飛星派的核心斷法：
1. 生年忌入何宮 → 先天課題所在
2. 大限忌入何宮 → 這十年的課題
3. 流年忌入何宮 → 今年的課題
4. 若三層忌都指向同一宮 → 該領域今年壓力極大，需要特別注意
5. 若忌入某宮又衝對宮 → 對宮領域受到間接影響

### 祿忌對沖
化祿所在宮位與化忌所在宮位如果是對宮關係，形成"祿忌對沖"：
- 代表該軸線上有明顯的矛盾和拉扯
- 比如祿在官祿忌在夫妻 → 事業和家庭之間的矛盾
- 祿在財帛忌在福德 → 賺到錢但精神壓力大
